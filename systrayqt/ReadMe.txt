This repository contains the cross platform Qt source and makefiles. This code
should be built with Qt Creator & Qt.

Windows build:
Configure Qt for: Windows Desktop, Qt 5.3 (currently build using MSVC20121 OpenGL 32 bit)

At present, the source path is assumed to be: C:\Qt-Projects\sysTrayQt
and the build path (for release) is: C:\Qt-Projects\build\WHATEVER
